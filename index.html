What's new in version 3.0.0?

Welcome back! Some major changes have occured around here, and you're now getting to experience them! Here's a list of changes we've made! • Redesigned the UI from the ground up to be both more asthetically pleasing and functional, while still maintaining low system usage. • Added the capability to easily add or remove different server types from the UI • Switched from .NET to Electron allowing for upgradability and functionality on nearly every platform • Added CPU and RAM usage monitoring • Added support for User accounts via a network-based database (not currently enabled) • Added the System Information panel, which allows users to see general system information as it may be needed in some circumstances. • Added the search function. It serves no purpose now, but as time progresses and we add more servers to the UI, this will come in handy. • Added the System Tools panel as a seperate page for Shutdown, Administrative Access, rebooting, etc. I hope you like this update, if you have any ideas, bug reports, or other requests, please direct them to Exile#6007 on Discord.
